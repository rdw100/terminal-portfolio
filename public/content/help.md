## Available Commands

- **about** — About Dusty
- **clear** — Clears the terminal
- **projects** — Displays GitHub projects
- **resume** — Displays an ASCII resume
- **socials** — Check out my social accounts
- **gui** — Opens the gui website in a new tab
- **lighthouse** — View latest Lighthouse scores
- **coin** — View select coin prices
- **welcome** — Displays hero section
- **theme** - Changes the terminal theme

<p>Tab autocompletes the command<br>
Up Arrow or Down Arrow to view previous commands<br>
ESC clears the terminal</p>
