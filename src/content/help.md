## Usage Examples

- `help` — show this help page  
- `projects` — list all projects  
- `projects goto 3` — open project #3  
- `socials` — list social links  
- `socials goto 2` — open social link #2  

---

Key **Tab** autocompletes the command<br>
Key **Up Arrow** or **Down Arrow** to view previous commands<br>
Key **ESC** clears the terminal
